Projetos

Mineração de Dados Educacionais de e-Learning: Concepção de Software para Acompanhamento do Aprendizado e Personificação do Ensino

Projeto de Pesquisa

O objetivo deste projeto é o desenvolvimento de estilos de interação que permitam o acompanhamento individualizado de alunos em atividades educacionais nos ambientes virtuais de aprendizagem a partir da aplicação de técnicas de EDM para análise de dados em plataformas educacionais. Nesta proposta pretendemos desenvolver: componentes de software para formações de grupo de estudo, sistema de indicação de material didático, componentes para os acompanhamentos educacionais, indicação de objetos de aprendizagem, predição de evasão de alunos, predição de performance e indicação de possíveis relacionamentos educacionais em plataformas de aprendizagem.

Design da Experiência de Desenvolvimento de Jogos Educacionais

Projeto de Pesquisa

Uma vez que os jogos eletrônicos além de exercerem um fascínio são um grande apelo para um aprendizado mais colaborativo divertido, lúdico e interativo. Este projeto pretende desenvolver um Design da Experiência de Desenvolvimento de Jogos Educacionais, buscando correlacionar características de jogos educacionais com seu potencial pedagógico, permitindo criar orientações de projetos em design de jogos educacionais.

Grupo de Pesquisa

O Grupo de Pesquisa em Ciência de Dados Educacionais (CiDE) integra pesquisadores de diversos departamentos (Educação, Computação, Psicologia) com foco no desenvolvimento e utilização de técnicas presentes em diversas áreas interdisciplinares, visando a extração de conhecimento a partir do processamento de grandes volumes de dados provindos de ambientes educacionais. Em nossas pesquisas buscamos converter dados brutos, de sistemas educacionais, em informações úteis que podem ser utilizadas por desenvolvedores de softwares educacionais, i essores, pesquisadores, gestores e interessados em educação. Estas informações são úteis e permitem um acompanhamento de alunos em atividades educacionais com uso de tecnologias. O grupo de pesquisa busca agregar as principais áreas: Educational Data Mining (EDM), Learning Analytics (LA), Educational Statistics (ES).
